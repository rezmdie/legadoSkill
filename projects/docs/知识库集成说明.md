### 2. MCP服务器调用前检查知识库

**位置**：[`src/mcp_server.py`](src/mcp_server.py:224)

**功能**：在调用分析工具前，自动搜索知识库获取相关知识。

**实现**：
```python
# 在调用分析工具前，先检查知识库并提供相关知识
knowledge_context = ""
if name in ["create_book_source", "analyze_website", "debug_book_source", "validate_selector"]:
    # 根据工具类型搜索相关知识
    query_map = {
        "create_book_source": "书源创建 规则编写",
        "analyze_website": "网站分析 选择器",
        "debug_book_source": "调试 规则验证",
        "validate_selector": "选择器 CSS XPath"
    }
    
    if name in query_map:
        try:
            # 调用知识搜索工具
            knowledge_result = await search_knowledge.ainvoke({
                "query": query_map[name],
                "top_k": 3
            })
            if knowledge_result and not knowledge_result.startswith("[ERROR]"):
                knowledge_context = f"\n\n## 📚 相关知识库参考\n\n{knowledge_result}\n\n---\n\n"
        except Exception as e:
            # 知识库查询失败不影响主流程
            pass
```

## 使用指南

### 自动知识库查询

当使用以下MCP工具时，系统会自动查询知识库：

1. **create_book_source** - 创建书源
   - 自动搜索：书源创建、规则编写相关知识
   
2. **analyze_website** - 分析网站
   - 自动搜索：网站分析、选择器相关知识
   
3. **debug_book_source** - 调试书源
   - 自动搜索：调试、规则验证相关知识
   
4. **validate_selector** - 验证选择器
   - 自动搜索：选择器、CSS、XPath相关知识

### 手动查询知识库

如果需要手动查询知识库，可以使用：

```python
# 使用 search_knowledge 工具
result = await search_knowledge.ainvoke({
    "query": "CSS选择器",

```

### 轻量级搜索

### ⚠️ 重要提醒

1. **知识库内容仅供参考**
   - 不能直接照搬知识库中的选择器
   - 所有选择器必须在真实HTML上验证
   - 禁止编造选择器

2. **必须验证**

   - 系统会继续执行原有功能
   - 错误会被静默处理
