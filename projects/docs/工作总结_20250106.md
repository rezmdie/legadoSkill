### 2. 创建书源JSON验证器

**文件**: `scripts/validate_book_source.py`

**功能**:
- ✅ 验证书源JSON格式
- ✅ 检查必填字段完整性
- ✅ 验证规则语法正确性
- ✅ 检查POST请求配置格式
- ✅ 生成详细的验证报告

**使用方法**:
```python
from scripts.validate_book_source import BookSourceValidator

validator = BookSourceValidator()
passed, errors, warnings = validator.validate(json_str)
validator.print_report()
```

**验证项目**:
1. JSON格式是否正确
2. 是否为数组格式
3. 是否包含所有必填字段
4. POST请求配置是否符合规范
5. 规则语法是否正确
